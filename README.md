# 🧠 ALETHEIA

**ALETHEIA** is a minimalist philosophical web experience designed to explore ideas through simplicity and clarity.

> *"L'index qui dévoile la vérité derrière les mots."*

---

## 📌 Overview

ALETHEIA is a lightweight front-end project that emphasizes:

* 🧘 Simplicity & Brutalist design
* 🔍 Concept-based search interaction
* ⚡ Fast performance (Lighthouse optimized)
* ♿ Accessible structure
* 🎨 Clean and consistent UI

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* Google Fonts & Icons
* Font Awesome Icons

---

## ⚡ Features

* 🔍 Search input interaction (JavaScript-based)
* 📱 Responsive design (mobile & desktop)
* 🎨 Minimalist UI inspired by philosophical themes
* 🧭 Navigation menu (Hero + Sections)
* 🖼 Optimized images (WebP format)

---

## ⚡ Performance Optimization

* Optimized images (`.webp`)
* Reduced unused CSS & JS
* Minified assets (recommended)
* Lazy loading (can be added)

---

## ♿ Accessibility

* Semantic HTML structure (`header`, `main`, `section`, `footer`)
* Readable typography
* Contrast-aware color usage
* Scalable layout

---

## 🔍 SEO

* Meta viewport configured
* Structured content hierarchy
* Meaningful headings (H1 → H3)

---

## 🎨 Design System

### 🎨 Colors

* **Primary Blue** → #009DFF
* **Obsidian Black** → #0D0D0D *(fix small typo from SVG fill)*
* **White / Light** → #FFFFFF

---

### 🔤 Typography

* Custom font: *Boldonse* (Google Fonts)
* Clean hierarchy:

  * H1 → Branding (ALETHEIA)
  * H2 → Tagline
  * H3 → Sections
  * P → Content
